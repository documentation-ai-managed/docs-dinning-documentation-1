## Upgrade Guide

Follow these steps to update your ดีเจ Dinning Documentation site. Test in a staging environment first.

<Steps>
  <Step title="Backup Your Site" icon="database">
    Create a full backup of your repository.

    ```bash
    git clone your-repo.git backup-docs
    ```
  </Step>
  <Step title="Update Package" icon="package">
    Install the latest version using your preferred package manager.
  </Step>
  <Step title="Verify Components" icon="check-circle">
    Run a build and check for parsing errors.
  </Step>
</Steps>

<Tabs>
  <Tab title="npm" icon="package">
    ```bash
    npm install @dj-dinning/docs@latest
    npm run build
    ```
  </Tab>
  <Tab title="Yarn" icon="package">
    ```bash
    yarn add @dj-dinning/docs@latest
    yarn build
    ```
  </Tab>
  <Tab title="pnpm" icon="package">
    ```bash
    pnpm add @dj-dinning/docs@latest
    pnpm build
    ```
  </Tab>
</Tabs>

<CodeGroup tabs="CLI,Docker">
  ```bash
  npx @dj-dinning/docs upgrade
  ```
  ```dockerfile
  docker pull dj-dinning/docs:latest
  docker run --rm -v $(pwd):/docs dj-dinning/docs:latest upgrade
  ```
</CodeGroup>
